# Pandas Data Extraction & Visualization

•  Project Overview
This project demonstrates data analysis using Python libraries such as Pandas, NumPy, Seaborn, and Matplotlib.

•  Features
- Data extraction from files using os and glob
- Data cleaning:
  - Removing NaN and null values
  - Replacing missing and blank values
- Data visualization:
  - Bar Chart
  - Column Chart
  - Histogram
  - Pie Chart

•  Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- OS & Glob

•  Project Workflow
1. Import required libraries  
2. Extract data from files  
3. Clean and pre-process data  
4. Perform data visualization

•  Conclusion
This project helps in understanding how to clean raw data and generate meaningful insights through visualizations.
